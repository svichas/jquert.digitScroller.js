# jquery.digitScroller.js
A jquery plugin for digit scrolling.

### Example
```html
<script>
  $(function() {

    $(".digit_scroll_container").digitScroll({
      scrollTo: 63, // number to scroll to
      changeDurr: 150 // durr between change
    });

  });
</script>

<div class="digit_scroll_container">00</div>
```
